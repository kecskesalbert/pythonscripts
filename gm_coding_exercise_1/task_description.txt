Task 1
Task description

Given a graph of n nodes. You are given an integer n and an array of edges   where edges[i] = [ai, bi] indicates that there is an edge between vertices ai and bi in the graph.    Write a function to compute the number of connected components in the graph.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Example 1:

 

   0

   |

   1 -- 2 -- 4    5 -- 6

   |

   3 

 

n = 7

edges = [[0, 1], [1, 2], [1, 3], [2, 4], [5, 6]]

returns 2

 

----------------------------------------------------

 

Example 2:

 

  0 -- 1    3 -- 4   5 -- 6

  |

  2

 

n = 7

edges = [[0, 1], [0, 2], [3, 4], [5, 6]]

returns 3

 

----------------------------------------------------

 

Example 3:

 

  0         1

  |         |

  4 -- 2 -- 3

 

n = 5

edges = [[0, 4], [4, 2], [2, 3], [3, 1]]

returns 1

----------------------------------------------------

 

Example 4:

 

    0

    |

    1 -- 7 -- 8    5 -- 6 -- 2 — 4

    |

    3 —- 9

 

n = 10

edges = [[0, 1], [7, 8], [3, 9] [5, 6], [2, 4], [1, 3], [7, 1], [6, 2]]

returns 2
